ActivUFRJ

Implementation of a new feature on the ActivUFRJ portal, using python/tornado and non-relational database(MongoDB).

ActivUFRJclose

My goal on this project was to develop a blog feature and its templates for an existing system called ActivUFRJ. The ActivUFRJ system is supposed to serve the UFRJ community as an education tool where teachers, researchers, staff and students can discuss and share interests, where they can pursue their own personal blog as well.

Songs

Single page React app, result of the 50% completion of a React-Redux course on udemy.

Songsclose

Songs is a very simple app just for showing and playing around with react and redux, while I keep making progress on the course. Basically I'm just showing a list of songs where I can select one of them and update the state of the app on the redux side of the application and also update the screen rendering the selected song details on the right. More sofisticated features like API requests, routing, etc will be available soon as I go further in the course.
